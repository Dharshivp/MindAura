# MindAura 🧠💙

## Overview

MindAura is an AI-based digital mental health support system designed to monitor and analyze students' emotional well-being. The system uses Natural Language Processing (NLP) and sentiment analysis to understand users’ emotions from text inputs and identify signs of stress, anxiety, or emotional distress.

When critical emotional patterns are detected, the system can alert counselors so that timely support and intervention can be provided.

## Features

* 🤖 AI-based emotion detection
* 💬 Real-time chat and sentiment analysis
* 👩‍⚕️ Counselor dashboard for monitoring students
* 📊 Emotion analysis charts and reports
* 🧑‍🎓 Student dashboard for interaction and support
* 🔔 Early alert system for critical emotional states

## Installation

1. Clone the repository

```
git clone https://github.com/Dharshivp/MindAura.git
```

2. Navigate to the project folder

```
cd MindAura
```

3. Install required dependencies

```
pip install -r requirements.txt
```

4. Run the application

```
python app.py
```

## Usage

* Students can log in and interact with the system through chat.
* The AI analyzes emotional tone from messages.
* Counselors can monitor students’ emotional patterns through the dashboard.

## Note

Due to GitHub file size limitations, the trained AI model file is not included in this repository.
Place the trained model inside the `saved_model` folder before running the project.
